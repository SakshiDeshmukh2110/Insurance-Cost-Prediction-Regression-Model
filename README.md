# Insurance-Cost-Prediction-Regression-Model
Performed exploratory data analysis (EDA) and built regression models to predict medical insurance charges based on factors like age, BMI, gender, region, and smoking habits. The project focuses on understanding variable impact, feature correlation, and model accuracy for cost prediction.

Project Overview

This project focuses on predicting medical insurance charges using regression techniques.
The goal was to analyze how demographic and lifestyle factors (age, BMI, smoking status, etc.) influence insurance costs and to build an accurate predictive model.
